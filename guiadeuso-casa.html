Guía de uso del sistema

Haz clic en las imágenes para ampliar

1. Dashboard (Pantalla Principal)

Es la pantalla de inicio donde se tiene una visión general rápida del estado financiero semanal.

  • Deudas vencidas: Detalle de pagos que ya pasaron su fecha.
  • Por vencer: Avisos con 7 días de anticipación para próximos pagos.
  • Pagado en la semana: Total acumulado de gastos ya abonados.
  • Saldo disponible: El dinero restante después de cubrir todos los pagos.
CLIC PARA AMPLIAR

2. Gastos y Boletas

En esta sección se generan las boletas por pagar o ya pagadas para mantener registro de cada una con su total y fecha de vencimiento.

Al cargar una boleta deberás registrar:

  • Categoría: A que corresponde.
  • Descripción: Algún detalle importante.
  • Monto: Valor del pago.
  • Fecha de vencimiento: Indicar la fecha de vencimiento o, en caso de estar pagado, la fecha de pago.
CLIC PARA AMPLIAR

Mantener estas boletas actualizadas permite que el sistema descuente los montos del presupuesto semanal.

3. Presupuesto

En esta categoría se asigna un presupuesto inicial, del cual se irá descontando el dinero a medida que se registren gastos. Todo ítem marcado como “Pagado” se descuenta automáticamente de ese presupuesto.

Configuración de ingresos:
Utiliza el botón "Configurar ingreso" para establecer el monto semanal con el que iniciarás el periodo.

Reglas principales:

  • Descuento automático: Todo lo que se marque como "Pagado" se descuenta del presupuesto.
  • Gestión de boletas: Si no haces clic en "Pagar", el gasto no se descuenta ni se archiva.
CLIC PARA AMPLIAR

4. Historial

Al finalizar cada semana, los datos de ese periodo se guardan y pasan automáticamente a esta sección. Es vital marcar los pagos para que el historial sea preciso al cerrar la semana.

CLIC PARA AMPLIAR
  • Las boletas cargadas que no se marquen como pagadas no aparecerán en el historial.
  • Esas boletas quedarán como gastos no pagos y se sumarán a la semana siguiente hasta abonarse.
